# SquadWork

SquadWork is a collaborative workspace designed to help teams organize tasks, track progress, and work together efficiently in one place. It focuses on simplicity, speed, and real-time collaboration so squads can spend less time managing work and more time getting things done.

## Features
- Team-based workspaces  
- Task creation, assignment, and status tracking  
- Real-time updates and collaboration  
- Clean, intuitive UI  
- Scalable backend-ready architecture  

## Tech Stack
- **Frontend:** React / Next.js  
- **Backend:** Node.js, Express  
- **Database:** PostgreSQL / MongoDB  
- **Auth:** JWT / OAuth  
- **Styling:** Tailwind CSS
